# Zynk User App

Customer-facing mobile application for Zynk Quick Commerce platform.

## Features
- Browse products by categories (Groceries, Fresh, Drinks, Pharmacy)
- Add items to cart and checkout
- Real-time order tracking
- Multiple payment options
- Order history and profile management

## Tech Stack
- React Native / Expo
- Supabase (Backend & Auth)
- React Navigation
- Redux Toolkit (State Management)

## Setup
```bash
npm install
npm start
```
